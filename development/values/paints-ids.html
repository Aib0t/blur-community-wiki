Car paints.

Paints (internal nameVEH_MaterialPaint are stored in vehicles.bin and consists of many values.

Paints can be divided in 2 categories:

  • Basic (available from color picker)
  • Special and car-related

It should be noted that some cars can't be painted because of textured body (wraps).

Paint object structure

Paints in Blur are very far from simple. Paints are stored as 1 pixel tall and 255 pixels wide dds texture with alpha channel. RGB channels used as diffuse map and alpha channel controls reflectivity of the surface.

Every paint contains:

  • undercoatColour (in RGB)
  • scuffColour (in RGB)
  • colours - array of RBG values
  • colourCurve - cubic hermit spline, defined by key frames
  • reflectivityCurve - cubic hermit spline, defined by key frames

You can check out example of exported paint structure at the bottom of the page.

Basic paints

Basic paints has distinctive names:

  • R2C5x - for basic color
  • M_R2C5x - for metallic version
  • P_R2C5x - for pearlessence variant
  • C_R2C5x - for candy variant

Name of basic paints consists of:

  1. Type header
  2. Row number
  3. Column number
  4. Shade letter (from a to c)

You can use a decoder below to for better understatement of where paint is located on in-game color picker.

Car-specific paints

All cars in game has a default paint job. This paint jobs consists of the same data as basic paint and can be applied to any other cars with some degree of success.

Special paints

Some special paints are referenced in game file. Gold, silver, chrome and some others. Currently untested. 

Please, report you results of applying paints from this category to Aibot#3399 at Discord.

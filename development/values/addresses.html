Blur.exe+D1BAD4 = camera max radius (float)
Blur.exe+D1B95C = camera max height (float)

Blur.exe+E35C5A = 01235C5A = Multiplayer Playlist Slots
If set to 1 and you are the only playere, e.g, 1/1 it will not start the match unfortunately,
but it does trigger the effect as in it is about to start, nonetheless part of the hud is gone and you wait endlessly

00BF65E4 = If NOP it increases the player slots available to 20
if set to a number ]0,20[ you get a prompt saying: Unable to create your multiplayer game session

Blur.exe+E35C59 = 01235C59 = Minimum Multiplayer Playlist Slots

["Blur.exe"+00DEFD58]+84 = Current Viewports

 

Executable functions 

Re-enables back button combos in Blur 1. 

Avalible features:

  • Slowmotion - Back + Y
  • Follow prev/next car on track - Back + Left/Right on D-pad

Changing the instruction here to PUSH 0x1 enables some back button combos (the button on the controller).

you can change the ingame camera to focus on different players and change your car to be AI controlled, I still can't get into the debug menu though

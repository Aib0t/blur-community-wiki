Structure description

The struct follows this scheme:

01 00 * 00 * 00 00 00 + UNK REFERENCE + OBJECT ID + OBJECT TYPE REFERENCE

Where:

  • UNK REFERENCE: Unknown reference possibly to another data type
  • OBJECT ID: The object id used by the gameâ€™s database to identify it
  • OBJECT TYPE REFERENCE: Used to reference all objects part of a single class
  • * : Wildcard. Can be multiple values. Unknown as of now what is its actual meaning

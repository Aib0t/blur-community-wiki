Launching Blur under Linux

Requirements:

  1. A linux-based system with Wine and PlayOnLinux installed;
  2. The game installation files, made available in the #installation-of-game and #blur-amax-emu-server section of the discord server (perform the entire step-by-step provided in the instructions of these channels before continuing);
  3. CPU: Intel i5-7200U;
  4. GPU: Intel HD graphics 620; and
  5. Memory: 8GB DDR4.

For hardware requirements, keep in mind that this does not reflect the actual minimum requirements, but rather the hardware that this procedure has been tested for.

Steps:

1. Run PlayOnLinux and select the "Install a program" option on the left of the window.

2. Look for "Blur" in the search box and select to install the only option that appears.

3. After that, continue with the installation as normal. When you arrive at the screen that asks for the installation method, select the option "Use a setup file in my computer".

4. Before choosing executable, move "Blur" folder containing the game files to ~/.playonlinux/wineprefix/BlurTheGame/drive_c/Program Files (x86)/

5. Rename the executable you want to use to "Blur.exe" (for example, if you want to use the  Amax emu server, rename "blur_amax.exe" to "Blur.exe").

5. Select the Blur.exe as the executable (present in the folder you just copied).  The game will start, wait for the menu to appear, then close it.

6. Continue the rest of the setup normally.

Extra tip! If you have Feral Gamemode installed on your system, you can turn it on for this game as follows (tested for GNOME desktop environment):

  1. Select the game icon from the PlayOnLinux menu, then choose "Create a shortcut" option.
  2. Desktop file have been created in your desktop folder, open it with a text editor and replace the content of the "Exec" variable to "gamemoderun playonlinux --run "Blur" %F".
  3. Save the file and move it to ~/.local/share/applications.
  4. Restart your computer to ensure that the changes are applied, after that you should find a shortcut of the game present in the menu of your system.

Known bugs:

  1. Poor performance.
  2. Bluetooth headsets are ignored by the game (with pipewire audio manager).

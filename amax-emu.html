Amax Emu

If you just want to get straight to “playing Blur online” part - join our Discord server.

Amax Emu was created as a direct replacement for currently working, but hard to access, Blur online service. 

Working features:

  • Profile saving
  • Leaderboards
  • Matchmaking
  • Friends
  • Invites
  • Private games

Upcoming features:

  • Friends challenges
  • Custom game modes
  • Profile pictures

Not planned to be supported features:

  • Facebook integration
  • Twitter integration
  • Photos upload

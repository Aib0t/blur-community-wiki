Amax Emu

If you just want to get straight to “playing Blur online” part - join our Discord server.

Amax Emu was created as a direct replacement for currently working, but hard to access, Blur online service. 

Working features:

  • Profile saving
  • Leaderboards
  • Matchmaking
  • Friends
  • Invites
  • Private games

Upcoming features:

  • Friends challenges
  • Custom game modes
  • Profile pictures

Not planned to be supported features:

  • Facebook integration
  • Twitter integration
  • Photos upload

How to connect

  1. In order to play on Amax Emu server you need a copy of Blur installed on your PC.
  2. After that you need to download patched Blur.exe, which you can grab here - https://drive.google.com/file/d/1VCk4vu8eFX0hEeq2fy6OjFmqgTyCf_aR/view?usp=sharing , bundled together with 1.2 update. Extract archive content to the root of Blur folder, replacing existing files.
  3. Create an account at https://amax-emu.com/
  4. Login in game, using your login and pass. If you previously have played on official Blur server I suggest that you back up your save file just in case.
  5. Play Blur, like it is 2010
